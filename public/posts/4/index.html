網站更新日誌 #1

2026-03-05

今天我花了不少時間調整部落格的細節。
圖說文字

一、NOW 頁面

首先是處理 Now 頁面的標題。我發現雖然在檔案開頭設定了標題，但網頁上卻沒顯示出來，後來發現是因為 Hugo 的內容頁需要手動在模板或內文裡加入標題。為了解決這個問題，我調整了 baseof.html 和 single.html，讓所有頁面都能自動抓取標題，並確保每個分頁的標題高度都完全一致。

二、版面固定

接著，我解決了版面跳動的問題。因為不同頁面的內容長度不同，導致底部的 Footer 會上下亂跑。我透過 CSS 的 Flexbox 設定，讓底部能乖乖黏在螢幕最下方，保持視覺上的穩定。

在字體美化方面，我覺得原本的標題太粗大，所以把 全站標題縮小 並減輕了字重，讓它看起來比較精緻。我也把原本拿掉的 發文日期 找了回來，但特別調整了它的樣式——將字體換成更有質感的襯線體，並把顏色調成淺灰色，讓它看起來安靜一點，不會搶走內文的焦點。

三、統一視覺

最後，我用一段簡潔的 CSS 統一了文章列表與內頁的日期格式，讓全站的視覺風格達成一致。

雖然中間一度遇到標題消失的小意外，但現在整個部落格的版面對齊了，字體也調整成我喜歡的樣子，感覺順眼多了。
