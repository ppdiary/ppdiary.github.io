架站大抄

2026-03-06

1. Markdown 非段落換行

  • 文末加 兩個半形空格
  • 文末加 反斜線 \
  • 文末加 <br>

2. 空格現身小白點

如何清楚知道有沒有成功加空格？

  1. 開啟 Command Palette 路徑：功能列 View > Command Palette 快捷鍵： Ctrl + Shift + P (Mac: Cmd + Shift + P)
  2. 搜尋輸入： Toggle Render Whitespace 效果：你會看到文字後面出現了兩個微小的 . .。 如果沒有看到兩個點，代表空白沒按對。

3. 改文章標題下的日期格式

路徑：layouts/_default/single.html 位置：{{ .Date.Format "2006-01-02" }}

4. 改段落之間的距離

路徑：layouts/_default/baseof.html 位置：.post-content p { margin-bottom: 2rem; }

5. 改標題間距

位置：h2, h3, h4 { margin: 30px 0px 10px 0px; } 主要是第一個數值 30px 影響和上一段的距離

6. 優先權

第一關： 你的專案根目錄 assets/（你擁有最高改寫權）。 第二關： 主題目錄 themes/xxx/assets/（這是主題的原廠設定）。
如果你在第一關（自己的 assets/）建立了一個同名檔案，Hugo 就會優先用你的，這就是為什麼我們可以輕鬆自定義外觀而不破壞主題原稿。

7. CSS

CSS 的全名是 Cascading Style Sheets（階層式樣式表）。「Cascading」的意思就是後面的規則會蓋掉前面的。
如果你在 extended.css 定義了標題是黑色，但在最後一趴（原本 baseof 的內容）定義了標題要有「莫蘭迪紫螢光筆」，寫在越後面的會越優先顯示。這就是為什麼我建議把你最心愛的自定義樣式放在檔案的最底部。
