如雷的掌聲  慢慢消逝在黑暗之中
燈光亮起  對著鏡頭的每一個臉孔洋溢著真心的笑容
換下團服  撤離樂器  一切都結束了
 
 


 
  
我很努力想打些什麼
可是我對著螢幕發呆了好一會兒
就是不知道該從何下筆 >> 不是打不出來  就是一發不可收拾 +.+
(所以才會拖到現在才有文章生出來  )
(另一部分原因是前幾天很不方便打網誌) 
 
 
故事好長好長....
好多讓人意想不到的奇妙過程
好多我找不到形容詞描述的心情
要親身體驗過後才會真正瞭解  沒經歷過的種種起伏  那感受...
 
 
情緒變化真的很大
從雀躍的最高點逐漸掉落
辛苦與挫折慢慢累積  到達飽和的深淵  然後爆發
一切釋放過後  在最後一刻衝上巔峰
得到的終究是發自內心的笑容   
 
 
 
 
第一天
抱著滿心期待走進社部
 
 
還沒進入狀況  就只拿到了譜 >> 開始組練!!
(我覺得應該要有幹部介紹小小的開場這樣呀  )
然後就在沒有搞清楚組練時間的狀況下  悠哉地翻譜...
慈宜剛翻完"陝北三韻"沒多久  我還在跟"夢蝶"的 page 7-8 左右 奮鬥中
此時...大會報告 :「開始合奏!!!」 
噢~箭在弦上...再多的  &  也沒有用了
所以兩個揚琴小孩的第一次合奏  就是在五線譜視奏 and 看了簡譜卻找不到音的情況下度過 ><
 
 
然後我忘了是誰說的....第一天就覺得好累 
哈哈 好像不少人都心有戚戚焉
是真的不輕鬆....
第一天就預言了未來兩週「組練只有2小時 + 每天坐著7小時以上 + 大曲子爆難練還是要撐下去」水深火熱的日子 
 
 
 
 
第二天第三天....每一天都是一樣的
生活規律得不得了欸!!
 
 
每天  比上學晚一小時但還是很早地起床
　→  和平常沒兩樣地搭公車到學校
　→  越早到練越多自由輕鬆的組練
　→  兩小時後大家會自動坐好開始合奏
　→  每天最溫馨甜蜜的午餐時間 
　→  再也沒有組練只有一長串驚險恐怖的合奏
　→  終於可以回家休息卻總是伴著滂沱大雨 or 細細小雨
 ( →  翻譜. 抄譜. 聽音樂看譜.... 常常有很多音樂會的"功課"要做 )
　→  上床睡覺一天就這麼過去了
 
 
就這樣一直循環兩個禮拜....
 
 
午餐時間是非常愜意的好時光 
大家一起吃便當一起聊天 so happy :)
而且都有好多有趣的事情呢 XD
然後我們都會去佔據旗隊的位置哈哈 (有一次一個一年級的經過居然跟我們說"學姊好"欸  )
 
   
然後我說....社部真的是悶悶悶爆了!!
尤其是處在"最佳(吹不到)風水地位"的揚琴和打擊
一個是居於中央的悶熱盆地
另一個大概就是很偏僻的背風山區吧
 
 
每次合奏都練到汗流浹背是怎麼回事 
社部冷氣根本就是騙人的居然還需要冷氣卡
電風扇的風不是被擋住就是傳不到我們這裡
最後還是萬能小扇子比較有用一點  可是它不敷使用 ><
 
 
就這樣
休息時能到外頭呼吸新鮮空氣都變成了享受 ^^
 
 
 
 
第五天
是驚心動魄晴天霹靂的轉捩點
 
 
從中午開始....
我的鳳尾竹就不美麗了 
 
 
我知道
絲竹的每一個人都是主角都很重要
我也知道
我們能練習能合奏的時間少之又少
我可以試想
學姊可能是擔心時間不夠了... 還有她想讓我們更好的那份心
 
 
我真的沒有做好
我真的還需要時間練習 
 
 
只是
那一天學姊的每一句話每一個字都好刺耳
我甚至感覺到我犯下了不應該犯的錯
是嗎? 我開始懷疑我自己
練合奏練不好  我總是以初學的身份安慰自己
集訓期間  組練這麼少  我又沒辦法在家練習(其實我很想啊...)
我可以把曲子都走過一遍 但練熟不是一兩次就行的
感覺上  有很多說不出的苦
但 真的是這樣嗎??
 
 
學姊的質問  喚起了心中的疑問
我的能力"應該"這麼差嗎?
我是不是還不夠努力?
到底是學姊高估了我的能力
還是我低估了自己用心的程度
到底是學姊的要求太高
還是我
真的不夠努力 (?)
 
 
沒有肯定的答案了
我無法反駁 :(
 
 
下午的"小時候"
我看到學姊失望又帶點不耐煩的表情
像是在問著:「這麼簡單的東西也不會嗎??」
中午的情緒累積到這裡....其實我已經潰堤了
我沒有辦法再去想什麼努力不努力
只有做好和做不好  就是這麼地現實
而我  沒有把它做好 
 
 
後來我都不知道在幹麻了
心緒很亂很亂
眼前是霧霧的一片
信心  正一點一滴被擊垮 
 
 
 
 
但其實還是有很深的堅持與熱愛 
 
 
而且經過一天心靈的摧殘 (←現在氣氛應該是凝重的...但我覺得這個詞讓人很想笑XD)
我覺得我真的要再更努力把它練好
 
 
我一直在想...一直在考慮...
因為畢竟比較有困難
但是我真的練不完  練不完了!!!
我看品慈面有難色  她也不知道該不該答應吧
對阿我當時也是鼓起很大的勇氣提出要求的欸 (我擔心被拒絕XD)
想不到雅暄學姊的答覆如此乾脆:「當然可以啊 :D」  噢都心花怒放了 
 
 
然後我要說....
我力氣真的沒有很大啦!! (小小聲...)
因為是喜歡的東西呀 :)  so...會有衝勁!! (而且那天衝勁特別大XD)
 
 
品慈都這樣到處散佈我是大力士的消息 
 
 
 
 
(我覺得應該會有人看不懂這一段....啊哈那就算了  )
 
 
  
  
從挫折的那一天開始
莫名地我就收集到了好多加油 
(迷:不莫名阿因為妳太明顯了~)
 
 
噢我真的超級無敵大大大感動 
你們一定不知道"加油"兩個字對我有多大的影響吧
它可以勝過千言萬語  就這麼單純的鼓勵  讓我知道有你們的支持與陪伴  就是我前進的動力喔~
 
 
小烏龜用兩個字得到了我的心
孝如的網誌看了心頭暖暖暖
慈宜同病相憐依然不停地幫我打氣
xx學姊的加油讓我又驚又喜 (我怕講錯名字啦... but 學姊在我心裡已經有大愛心了  )
還有親朋好友 and 路人甲乙丙丁給予的支持  不論是對我或對這個音樂會
 
 
大家的加油  都會化成我跟挫折奮戰的力量!!  謝謝你們 :)
 
 
 然後就是....
千言萬語說不盡  無法言喻只能心領  一言一行都看在心裡  好感動也好感謝 的
子云學姊 & 品慈慈慈~~   
噢我超愛妳們的啦 >///<
 
 
謝謝子云學姊在不知道怎麼得到消息後就來留言關心我
謝謝子云學姊早上都會抽空來看我練習教我揚琴
謝謝子云學姊在我崩潰的時候給了我一個大大的依靠
 
 
謝謝品慈被我嚇到後打了一整篇滿滿的鼓勵文章
謝謝品慈always陪在我身邊默默地支持我幫助我
謝謝品慈從以前到現在都對我這麼好
  
 
我感受到了  都感受到心底了 :)
 
 
另外
謝謝那一天真的好兇嚇到我但其實人很好會講笑話的青雲學姊
謝謝學姊也是在不知道怎麼得到消息後過來安慰我跟我說不要太在意
 
 
安定多了
心情也慢慢調適好
 
 
(只是後來進步的空間仍然有限  讓我覺得很對不起學姊還有合奏的大家ˊˋ)
 
 
一個兩週的集訓
讓我發現....
松筠有好多好多充滿大愛心的人 :D
(怎麼辦這樣我喜歡的人越來越多了 >//<)
 
 
還有就是....
　
　
　
　
　
　
　
　
　
　
我發現我哭點整個就變低了啦 
　
　
 
 
我們換下一篇吧 
不然感覺好長噢~
(迷:已經很長了!!!)
