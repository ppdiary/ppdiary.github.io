## Markdown 非段落換行
* 文末加 兩個半形空格  
* 文末加 反斜線 \  
* 文末加 `<br>`

## 空格現身小白點
如何清楚知道有沒有成功加空格？

1. 開啟 Command Palette

    路徑：功能列 View > Command Palette  
快捷鍵： Ctrl + Shift + P (Mac: Cmd + Shift + P)

2. 搜尋輸入： Toggle Render Whitespace

    效果： 你會看到文字後面出現了兩個微小的 . .。  
如果沒有看到兩個點，代表空白沒按對。
