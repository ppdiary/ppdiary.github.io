> 以防未來的我忘記步驟。

## 第一步：建檔 & 寫文章
1. 開啟 VS code
2. Ctrl + ` 叫出終端機
3. 在終端機輸入 `hugo new posts/檔名.md`
 
 
## 第二步：預覽確認畫面

1. 在終端機輸入 `hugo server -D`
2. 開啟連結 http://localhost:1313
3. 修改 或 按下 Ctrl + S 存檔

## 第三步：公開發布到網路
1. 文章標頭 `draft: true` 改為 `draft: false`
2. 執行三個指令：  
` git add .`  
`git commit -m "一段描述，寫更新說明或碎碎念都可以"`  
`git push origin main`
