### 建檔與寫文章
1. 開啟 VS code
2. Ctrl + ` 叫出終端機
3. 在終端機輸入 `hugo new posts/檔名.md`
 --- 


### 在自己的電腦內預覽

1. 在終端機輸入 `hugo server -D`
2. 開啟連結 http://localhost:1313
3. 修改 或 按下 Ctrl + S 存檔
 ---

### 公開發布到網路上
1. 文章標頭 draft: true 改為 draft: false
2. 執行三個指令：
- git add .
- git commit -m "這裡可以寫本次更新的說明"
- git push origin main



---
### Hugo 的核心運作邏輯：
- 指令建檔 (hugo new)
- Markdown 寫作 (在 content 裡打字)
- 即時預覽 (hugo server)
