我都用 Toggl 記錄練琴時數，
今天終於實現透過 API 自動化抓資料的願望！！

這也是我第一次接觸 Python，
從把它安裝至電腦開始。

感謝 G大 一步步教會我：
- Python 是什麼：自動化搬運工，能定時去外面抓資料回來更新
- Python 環境安裝：蓋一間自動化辦公室
- 外掛安裝　`pip install requests`：幫 Python 買好「電話機」，讓它可以準備幫我去 Toggl 總部問資料　
- 執行 `python scripts/update_toggl.py` （即將自動化）
- 除錯 update_toggl.py.txt：副檔名.txt 需徹底刪除
- 除錯 避免時差問題 改用 ISO 8601 格式
- 10天內 改為以週為單位統計



完整的 「API 數據自動化管線」：

數據源：Toggl (妳的練習紀錄)

處理器：Python (妳的小幫手腳本)

呈現端：Hugo 網站 (妳的鋼琴日記)


🎓 妳今天完成的成就清單：
- 打通 API：讓 Python 能夠代表妳去 Toggl 總部搬資料。

- 精準過濾：只抓取「快樂練琴時光」，不讓工作雜事混入琴房。

- 時區對齊：解決了靜態網站最頭痛的「未來文章」隱藏問題。

- 內容自動生成：以後只要練完琴跑一下腳本，網頁就更新好了。
