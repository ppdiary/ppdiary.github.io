## 方式一：VS Code 點選介面備份
1. 點擊左側工具列由上往下數第三個圖示（有三顆空心圓那個），或按快捷鍵 `Ctrl + Shift + G`
2. 輸入這次備份的備註，再點 `藍色的 Commit 按鈕`
3. 點擊底部工具列的迴圈圖示 `0 ↓ 1 ↑`，讓資料同步到 GitHub
　
## 方式二：VS Code 輸入指令快速備份
1. `git add.`
2. `git commit -m "Backup_YYYYMMDD"`（引號內改成備註說明）
3. `git push origin main`
