## VS Code 點選介面備份
1. 左側工具列點開第三個（有三顆空心圓的圖示），或按快捷鍵 `Ctrl + Shift + G`
2. 輸入這次備份的備註，點 藍色的 Commit 按鈕
3. 點擊底部工具列的迴圈圖示 `0 ↓ 1 ↑`，讓資料同步到 GitHub  
　
## VS Code 輸入指令快速備份
1. 執行三個指令：
- git add .
- git commit -m "Backup_YYYYMMDD"  `引號內可改備註說明`
- git push origin main
